Use a **standard, non-dramatic** code of conduct. This is not about virtue signaling; it’s about professionalism.

```markdown
# Code of Conduct

## Our Pledge

We are committed to maintaining a respectful, inclusive, and professional environment for everyone contributing to or using this project.

This project exists to support scientific research and collaboration. All participants are expected to engage constructively and in good faith.

---

## Expected Behavior

- Use respectful and professional language
- Be considerate of differing viewpoints and experience levels
- Accept constructive criticism
- Focus discussions on scientific and technical merit
- Credit others’ work appropriately

---

## Unacceptable Behavior

- Harassment, discrimination, or hostile behavior
- Personal attacks or inflammatory language
- Misrepresentation of data or results
- Deliberate disruption of collaboration or review processes

---

## Scope

This Code of Conduct applies to:
- GitHub issues and pull requests
- Code reviews and discussions
- Documentation contributions
- Any communication related to this project

---

## Enforcement

Project maintainers are responsible for interpreting and enforcing this Code of Conduct.  
Unacceptable behavior may result in warnings, contribution restrictions, or removal from the project.

---

## Reporting

Concerns or violations should be reported privately to the project maintainers.

---

## Acknowledgement

This Code of Conduct is adapted from common open-source community standards and tailored for an academic research context.
